# Pawzen Website

Pawzen is a simple pet-related website designed to provide information and services for pet lovers. This project focuses on building a clean and responsive user interface using basic web technologies.

## Features
- Responsive design
- User-friendly interface
- Informational sections for pet care
- Clean layout using HTML, CSS, and JavaScript

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Run
1. Clone the repository
2. Open index.html in your browser
